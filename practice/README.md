# Practice Workbooks

This folder contains English practice material that matches the theory modules in `modules/`.

The exercises are grouped by track so learners can practice several related modules in one place while still following the original curriculum.

## Practice Tracks

1. [Track 1 Practice: Basic Python](./01-basic-python-practice.md)
2. [Track 2 Practice: Multi-File and Built-ins](./02-multi-file-and-builtins-practice.md)
3. [Track 3 Practice: Developer Environment](./03-developer-environment-practice.md)

## How to Use These Practice Files

1. Finish the matching theory lessons first.
2. Answer the written questions without looking back immediately.
3. Write the coding exercises in your own editor.
4. Review your answers after you finish a whole section.

## Goal

The purpose of this folder is to move from:

- reading concepts
- to explaining concepts
- to writing code with those concepts

That transition is where real understanding starts to become skill.
