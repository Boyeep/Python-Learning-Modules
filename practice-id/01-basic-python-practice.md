# Latihan Track 1: Basic Python

Workbook ini mencakup latihan untuk:

- Modul 1: Variables dan Data Types
- Modul 2: Operator dan Manipulasi String
- Modul 3: Conditional dan Control Flow
- Modul 4: Data Structure dan Function
- Modul 5: Error Handling

## Latihan Modul 1: Variables dan Data Types

### Pertanyaan Cepat

1. Apa perbedaan nama variabel dan value yang dirujuknya?
2. Kenapa `"42"` berbeda dari `42`?
3. `None` biasanya merepresentasikan apa?

### Tugas Coding Singkat

1. Buat variabel untuk nama siswa, umur, IPK, dan status aktif.
2. Cetak tipe dari setiap variabel dengan `type()`.
3. Ubah salah satu variabel menjadi tipe lain lalu jelaskan apa yang berubah.

### Refleksi

- Tipe data mana yang paling mudah kamu pahami saat ini?
- Tipe data mana yang masih terasa membingungkan?

## Latihan Modul 2: Operator dan Manipulasi String

### Pertanyaan Cepat

1. Apa perbedaan `=` dan `==`?
2. Apa yang dikembalikan oleh `%`?
3. Kenapa f-string berguna?

### Tugas Coding Singkat

1. Buat dua angka lalu cetak hasil penjumlahan, pengurangan, perkalian, dan sisanya.
2. Buat nama lengkap dari variabel nama depan dan nama belakang.
3. Ubah satu kalimat menjadi uppercase, lowercase, dan title case.
4. Buat satu kalimat dengan f-string yang berisi nama dan umur.

### Refleksi

- Kapan formatting string terasa lebih jelas daripada penggabungan string biasa?

## Latihan Modul 3: Conditional dan Control Flow

### Pertanyaan Cepat

1. Apa itu control flow?
2. Kapan kamu akan memakai `for` dibanding `while`?
3. Sebutkan tiga value falsy di Python.

### Tugas Coding Singkat

1. Tulis program yang mencetak apakah sebuah angka positif, nol, atau negatif.
2. Tulis aturan nilai menggunakan `if`, `elif`, dan `else`.
3. Gunakan `for` loop untuk mencetak setiap karakter dalam sebuah kata.
4. Gunakan `while` loop untuk menghitung dari 1 sampai 5.

### Refleksi

- Bentuk percabangan mana yang paling mudah kamu nalar?

## Latihan Modul 4: Data Structure dan Function

### Pertanyaan Cepat

1. Kapan list lebih baik daripada set?
2. Kapan dictionary lebih baik daripada list?
3. Apa perbedaan parameter dan return value?

### Tugas Coding Singkat

1. Buat list berisi lima topik favorit dan cetak item pertama serta terakhir.
2. Buat set dari value yang berulang lalu lihat hasilnya.
3. Buat dictionary yang mendeskripsikan buku dengan title, author, dan year.
4. Tulis function `greet(name)` yang mengembalikan sapaan.
5. Tulis function `average(a, b, c)` yang mengembalikan rata-rata tiga angka.

### Refleksi

- Data structure mana yang terasa paling natural bagimu?
- Data structure mana yang terasa paling powerful?

## Latihan Modul 5: Error Handling

### Pertanyaan Cepat

1. Apa itu exception?
2. Kenapa penanganan error yang spesifik biasanya lebih baik daripada yang terlalu luas?
3. Apa tujuan dari `finally`?

### Tugas Coding Singkat

1. Minta user memasukkan angka lalu konversi dengan aman memakai `try` dan `except`.
2. Tulis function yang me-raise `ValueError` ketika input kosong.
3. Buat contoh kecil yang memakai `try`, `except`, `else`, dan `finally`.

### Refleksi

- Menurutmu error seperti apa yang paling sering terjadi pada pemula?
