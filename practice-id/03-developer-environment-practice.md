# Latihan Track 3: Developer Environment

Workbook ini mencakup latihan untuk:

- Modul 10: Menyusun Struktur Project Python
- Modul 11: Python Packages dan Environment

## Latihan Modul 10: Menyusun Struktur Project Python

### Pertanyaan Cepat

1. Kenapa struktur project makin penting saat code bertambah besar?
2. Kenapa test sebaiknya punya tempat yang jelas?
3. Hal apa yang seharusnya mudah dikenali dari layout project yang baik?

### Tugas Singkat

1. Buat rancangan struktur folder untuk aplikasi todo berbasis terminal.
2. Tentukan di mana kamu akan menaruh:
   - entry point
   - helper function yang reusable
   - test
   - dokumentasi
3. Jelaskan kenapa strukturmu masuk akal.

## Latihan Modul 11: Python Packages dan Environment

### Pertanyaan Cepat

1. Masalah apa yang diselesaikan virtual environment?
2. Kenapa instalasi global berisiko?
3. Kapan Conda bisa lebih berguna daripada `pip` plus `venv`?

### Tugas Singkat

1. Tulis command yang akan kamu gunakan untuk membuat virtual environment.
2. Tulis command untuk meng-install package dengan `pip`.
3. Jelaskan kenapa file dependency penting untuk kolaborasi.

## Mini Challenge

Rancang setup plan untuk project Python pemula bernama `study_tracker`.
