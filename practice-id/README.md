# Workbook Latihan

Folder ini berisi materi latihan versi Bahasa Indonesia yang dipasangkan dengan modul teori di `modules-id/`.

Latihan dikelompokkan per track agar learner bisa berlatih beberapa modul yang saling berhubungan dalam satu tempat, sambil tetap mengikuti kurikulum utamanya.

## Track Latihan

1. [Latihan Track 1: Basic Python](./01-basic-python-practice.md)
2. [Latihan Track 2: Multi-File dan Built-ins](./02-multi-file-and-builtins-practice.md)
3. [Latihan Track 3: Developer Environment](./03-developer-environment-practice.md)

## Cara Menggunakan File Latihan Ini

1. Selesaikan dulu lesson teori yang sesuai.
2. Jawab pertanyaan tertulis tanpa langsung melihat ke materi.
3. Kerjakan coding exercise di editor milikmu sendiri.
4. Tinjau jawabanmu setelah menyelesaikan satu bagian penuh.

## Tujuan

Tujuan folder ini adalah membantu perpindahan dari:

- membaca konsep
- menjadi menjelaskan konsep
- lalu menulis code dengan konsep tersebut

Perpindahan itu adalah titik di mana pemahaman mulai berubah menjadi skill.
