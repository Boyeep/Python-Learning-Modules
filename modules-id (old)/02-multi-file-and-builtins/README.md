# Track 2: Multi-File dan Built-ins

Track ini mengajarkan bagaimana code Python berkembang melewati satu file.

Perubahan ini penting karena software nyata tidak ditulis sebagai satu script panjang saja.
Saat project membesar, kamu butuh:

- pemisahan tanggung jawab
- module yang bisa dipakai ulang
- kebiasaan dokumentasi
- kemampuan menangani file
- pemahaman yang lebih baik terhadap built-in tools Python

## Lesson dalam Track Ini

1. [Modules dan `__main__`](./01-modules-and-main.md)
2. [Docstring](./02-docstrings.md)
3. [Python Standard Library](./03-python-standard-library.md)
4. [File I/O](./04-file-io.md)

## Yang Harus Kamu Pahami Setelah Track Ini

- bagaimana import di Python bekerja
- kenapa code dibagi ke beberapa file
- bagaimana docstring membantu maintainability
- bagaimana standard library menghemat waktu
- bagaimana program membaca dan menulis file dengan aman
