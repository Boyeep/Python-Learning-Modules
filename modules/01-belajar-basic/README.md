# Track 1: Belajar Basic

This track teaches the foundation of Python programming.

The purpose is not only to help you write code, but to help you think clearly about:

- values
- logic
- flow
- reusable behavior
- failure handling

## Lessons in This Track

1. [Variables and Data Types](./01-variables-and-data-types.md)
2. [Operators and String Manipulation](./02-operators-and-string-manipulation.md)
3. [Conditionals and Control Flow](./03-conditionals-and-control-flow.md)
4. [Data Structures and Functions](./04-data-structures-and-functions.md)
5. [Error Handling](./05-error-handling.md)

## What You Should Understand After This Track

- how Python stores information in variables
- how different data types behave
- how expressions produce values
- how code makes decisions
- how repetition and function calls reduce duplication
- how programs fail and how to handle failure safely

## Core Mental Model

A Python program is a sequence of instructions that:

1. stores values
2. transforms values
3. makes decisions
4. groups logic
5. reacts to unexpected situations

If you understand those five things, you understand the foundation of programming.
