# Track 3: Developer Environment

Track ini mengajarkan cara bekerja pada project Python dengan lebih rapi dan maintainable.

Belajar syntax hanyalah permulaan.
Developer yang baik juga perlu memahami:

- struktur project
- environment
- dependency
- kebiasaan instalasi dan setup

- kenapa struktur project itu penting
