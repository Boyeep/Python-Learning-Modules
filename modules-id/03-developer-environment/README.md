# Track 3: Developer Environment

Track ini mengajarkan cara bekerja pada project Python dengan lebih rapi dan maintainable.

Belajar syntax hanyalah permulaan.
Developer yang baik juga perlu memahami:

- struktur project
- environment
- dependency
- kebiasaan instalasi dan setup

## Lesson dalam Track Ini

1. [Menyusun Struktur Project Python](./01-structuring-a-project.md)
2. [Python Packages dan Environment](./02-python-packages-and-environments.md)

## Yang Harus Kamu Pahami Setelah Track Ini

- kenapa struktur project itu penting
- bagaimana developer memisahkan code, test, dan konfigurasi
- apa itu package
- kenapa virtual environment penting
- kapan `pip`, `venv`, atau Conda berguna
