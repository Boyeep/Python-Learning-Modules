# Contributing

Terima kasih sudah berminat berkontribusi pada project ini.

## Alur Kontribusi

1. Fork repository.
2. Buat branch baru dengan nama jelas, misalnya `feature/add-readme`.
3. Jalankan test dan linting sebelum membuat pull request.
4. Buat pull request ke branch `main`.

## Checklist sebelum PR

- [ ] Kode sudah dijalankan dan diuji
- [ ] Test lokal sudah lulus
- [ ] Format kode sudah sesuai dengan aturan proyek
- [ ] Dokumentasi diperbarui bila perlu

## Coding Style

Ikuti konfigurasi formatter dan linter yang ada di project.
