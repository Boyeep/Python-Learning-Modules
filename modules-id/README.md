## Urutan Belajar yang Disarankan

1. Selesaikan Track 1 terlebih dahulu.
2. Lanjut ke Track 2.
3. Selesaikan Track 3 setelah kamu mulai paham syntax dasar dan penggunaan file di Python.

## Cara Belajar

Untuk setiap lesson:

1. Baca bagian teori dengan pelan.
2. Ketik ulang contoh kodenya sendiri.
3. Jawab prompt latihan tanpa langsung melihat ke atas.
4. Gunakan checklist di akhir untuk mengecek apakah konsepnya sudah masuk.

## Tujuan Utama

Jangan belajar Python hanya sebagai kumpulan syntax.

Pelajari Python sebagai sistem ide:

- bagaimana value disimpan
- bagaimana logika diekspresikan
- bagaimana code diorganisasi
- bagaimana program berinteraksi dengan file dan package
- bagaimana developer menjaga project tetap rapi seiring bertambah besar
