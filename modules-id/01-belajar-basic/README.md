# Track 1: Belajar Basic

Track ini mengajarkan fondasi pemrograman Python.

Tujuannya bukan hanya supaya kamu bisa menulis code, tetapi juga supaya kamu bisa berpikir lebih jelas tentang:

- value
- logika
- alur eksekusi
- perilaku yang bisa dipakai ulang
- cara menangani kegagalan

## Lesson dalam Track Ini

1. [Variables dan Data Types](./01-variables-and-data-types.md)
2. [Operator dan Manipulasi String](./02-operators-and-string-manipulation.md)
3. [Conditional dan Control Flow](./03-conditionals-and-control-flow.md)
4. [Data Structure dan Function](./04-data-structures-and-functions.md)
5. [Error Handling](./05-error-handling.md)

## Yang Harus Kamu Pahami Setelah Track Ini

- bagaimana Python menyimpan informasi ke dalam variabel
- bagaimana tipe data yang berbeda berperilaku
- bagaimana expression menghasilkan value
- bagaimana code membuat keputusan
- bagaimana pengulangan dan function mengurangi duplikasi
- bagaimana program gagal dan bagaimana menanganinya dengan aman

## Mental Model Inti

Sebuah program Python adalah urutan instruksi yang:

1. menyimpan value
2. mengubah value
3. membuat keputusan
4. mengelompokkan logika
5. merespons situasi yang tidak terduga

Kalau lima hal ini sudah kamu pahami, berarti fondasi programming-mu sudah mulai kuat.
