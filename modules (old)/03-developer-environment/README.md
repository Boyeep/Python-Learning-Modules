# Track 3: Developer Environment

This track teaches how to work on Python projects in a maintainable way.

Learning syntax is only the beginning.
A real developer also needs to understand:

- project structure
- environments
- dependencies
- installation habits

## Lessons in This Track

1. [Structuring a Python Project](./01-structuring-a-project.md)
2. [Python Packages and Environments](./02-python-packages-and-environments.md)

## What You Should Understand After This Track

- why project structure matters
- how developers separate code, tests, and configuration
- what packages are
- why virtual environments matter
- when `pip`, `venv`, or Conda are useful
