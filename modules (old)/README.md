## Suggested Study Order

1. Finish Track 1 completely.
2. Continue to Track 2.
3. Finish Track 3 after you already understand basic Python syntax and files.

## How to Study

For each lesson:

1. Read the theory sections slowly.
2. Type the example code yourself.
3. Answer the practice prompts without looking back.
4. Use the checklist at the end to see if the concept is clear.

## Main Goal

Do not study Python as a list of syntax rules.

Study it as a system of ideas:

- how values are stored
- how logic is expressed
- how code is organized
- how programs interact with files and packages
- how developers keep projects maintainable over time
