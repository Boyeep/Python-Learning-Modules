## What This Repo Contains

This repository is organized as a small curriculum.

It covers three main tracks:

1. Basic Python foundations
2. Multi-file Python and built-in tools
3. Python developer environment and project setup

Each track is split into smaller Markdown lessons so the material stays readable and easier to study over time.

| Track | Main Topic | Key Lessons |
| --- | --- | --- |
| Track 1 | Basic Python | variables and data types, operators and strings, conditionals and control flow, data structures and functions, error handling |
| Track 2 | Multi-File and Built-ins | modules and `__main__`, docstrings, Python standard library, file I/O |

## Recommended Study Order

1. Finish Track 1 first.
2. Continue to Track 2.
3. Study Track 3 after you already understand the basics of Python syntax and files.

## How to Use This Repo

For each lesson:

1. Read the theory section slowly.
2. Type the example code yourself instead of only reading it.
3. Answer the practice prompts.
4. Check the mastery checklist before moving on.

## Who This Repo Is For

This repo is useful for:

- beginners who want stronger Python fundamentals
- students who want theory modules, not only random notes
- self-learners who want both English and Indonesian material
- learners who want a clean study path before starting bigger Python projects

## Main Philosophy

The purpose of this repo is not just to teach you how to write Python code.

It is meant to help you understand:

- how values are stored
- how logic flows
- how programs are organized
- how files and modules work
- how real Python projects are structured

That kind of understanding is what makes future learning much easier.
